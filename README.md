# 📊 Calculador de Promedios

Un programa simple y práctico en Python para calcular el promedio de una lista de números.

## 🎯 ¿Qué hace?

Este script te permite:
- Ingresar números (enteros o decimales) de forma interactiva
- Calcular automáticamente el promedio
- Mostrar el resultado de manera clara

## 🚀 Cómo usar

### Requisitos
- Python 3.x

### Ejecución

```bash
python3 promedio.py
```

Luego ingresa los números separados por espacios:

```
Ingrese números separados por espacios, o presione Enter para terminar:
> 10 20 30 40 50
Promedio: 30.0
```

## 💡 Características

✅ Entrada interactiva de números  
✅ Soporta enteros y decimales  
✅ Validación de datos  
✅ Mensajes de error claros  

## 📝 Notas

- Presiona **Enter sin escribir nada** para terminar la entrada
- Los números inválidos se ignoran automáticamente
- Es necesario ingresar al menos un número válido
